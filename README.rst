
seqtools2
=========

Bioinformatic tools for manipulation, filtering, and statistics

based on seqtools

In honor of John A. Crow

Current Tools
-------------
USAGE: sequencetools <TOOL> [options]

Current Tools:

  basic_fasta_stats
  format_fasta
  chunk_fasta
  chunk_fastq
  fastx_converter
  filter_fasta_by_length
  get_fasta_by_id
  get_fastq_by_id
  subset_fastq

Please run `sequencetools <TOOL> --help` for individual usage
